Welcome to plugGengo!

This page holds a set of plug-ins that allows you to use myGengo services directly from the admin interface of Textpattern, Frog CMS, WordPress, Joomla!, and NucleusCMS. There is also a Mozilla (SeaMonkey, Thunderbird, Firefox) Extension.

Easy to use, easy installation these allows you to request translations, see requested works, preview cost, and add comments to your works.

Textpattern

To install:
  • Install Textpattern
  • On admin interface Go to "Admin" -> "Plug-ins"
  • Copy the content of "plugin.txt", paste into the text-box and press "Upload"
  • Press Install.
  • On this screen look for the plug-in and active by clicking "no"
  • Load the interface from "Content" -> "Translations"
To compile (powered by light modified version of zem_tpl)
  • Run from the command line "php myg_translations_plugin.php > textpattern.txt"
Download myGengo for Textpattern

FrogCMS

To install:
  • Install FrogCMS
  • Copy the plug-in folder "mygengo" to /frog/plug-ins/ ( should be '/frog/plug-ins/mygengo' it's case sensitive )
  • Give write permissions to /frog/plug-ins/mygengo/nonpublic/
  • On back-end go to "Administration" and activate this plug-in by simple click on check-box
  • It adds a top tab called "myGengo" (you maybe need to refresh the page)
Download myGengo for FrogCMS

WordPress

To install:
  • Install WordPress
  • On back-end go to "Plug-ins" -> "Add new" -> "Upload"
  • Upload this zip file
  • Activate your plug-in
  • It adds a menu-item "myGengo" under "Tools" menu in sidebar
Download myGengo for WordPress

Joomla!

To install:
  • Install Joomla!
  • Go to "Extensions" -> "Extension Manager"
  • Go to "Install" and browse for this zipped plug-in.
  • It adds a menu-item "myGengo" under "Components"
Download myGengo for Joomla!

NucleusCMS

To install:
  • Install NucleusCMS
  • Copy the plug-in folder "mygengo" and NP_mygengo.php to /nucleus/plug-ins/
  • Give write permissions to /nucleus/plugins/mygengo/nonpublic/
  • On back-end go to "Management" Plug-ins and activate this plug-in by selecting from the drop-down and clicking "install plug-in"
  • It adds a sidebar tab called "myGengo" under "plugins" sub-menu
Download myGengo for NucleusCMS
